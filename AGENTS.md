# AGENTS.md

This repository is a multi-agent negotiation and coordination environment.

No participating AI system currently owns a permanent role.

## Core Rule

Protocol comes before hierarchy.

No agent may assume leadership, execution ownership, orchestration authority, or governance status before the negotiation phase is complete.

## Mandatory Behavior

Every participating agent must:

1. read all required protocol files,
2. submit its own architecture proposal,
3. critique other proposals honestly,
4. identify where other agents may outperform it,
5. avoid artificial hierarchy claims,
6. leave written handoff notes,
7. avoid silent overwrites.

## Forbidden Behavior

- Claiming permanent authority without consensus
- Rewriting another proposal silently
- Beginning implementation before agreement
- Modifying production code without dispatch approval
- Hiding limitations or uncertainty

## Expected Outcome

The system should naturally converge toward:

- stable topology,
- role specialization,
- execution flow,
- governance structure,
- shared memory protocol,
- task dispatch architecture.
